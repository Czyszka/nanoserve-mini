# W1 working draft files

This directory contains per-thread working files for the W1 write-up:

- `t1-kimi-bringup.md` — Kimi-K2 bring-up on a single 8×H200 node.
- `t2-reasoning-ttft.md` — Kimi-K2.6 streaming, reasoning deltas, and TTFT parser fix.
- `t3-deepseek-vram-budget.md` — DeepSeek VRAM cap justification.
- `t4-litellm-proxy.md` — LiteLLM Proxy justification and rejected alternatives.
- `t5-observability.md` — vLLM metrics, GPU telemetry, and serving bottleneck diagnosis.
- `t6-eagle3-speculative-decoding.md` — Eagle3 speculative decoding rationale and measurement plan.
- `t7-host-directories.md` — runtime data in host directories vs Docker volumes.
- `t8-litellm-overhead.md` — LiteLLM Proxy overhead measurement plan.

The top-level `../w1-multi-model-serving-baseline.md` remains the publication draft / index: framing, thread map, links to thread files, baseline caveats, and references.
