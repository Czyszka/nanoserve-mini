Standalone scripts
request_once.py
Single non-streaming request — smoke / correctness
mode: singlestream_lite_correctness
out: e2e_s · output_chars · tokens
measure_ttft_once.py
Single streaming request — TTFT / E2E / TPOT
mode: singlestream_lite_latency
out: ttft_s · ttft_any_token_s · tpot_s
run_sequential_benchmark.py
N sequential streaming requests, p50/p95 summary
mode: singlestream_lite_repeated
out: summary.json + results.jsonl
collect_metrics_snapshot.py
One-shot vLLM /metrics + nvidia-smi snapshot
out: gpu_mem_gb · kv_cache · prefix_hit
sample_gpu_metrics.py
Interval CSV sampling: util, mem, power, clocks
out: gpu_samples.csv + _meta.json
check_server_env.py
System inventory: git, uv, nvidia-smi, docker…
out: server_env_snapshot.json
